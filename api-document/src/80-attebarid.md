# 关注网吧

* POST `/capi.php`
* c `80`

## 请求参数

| 参数名 | 参数类型 | 参数说明 | 备注 |
| :---- | :----| :----| :---- |
| sid | string | sessionid | 登录成功后返回 |
| id | string | 用户id | 登录成功后返回 |
| is_atte | int | 0:取消，1：关注 |
| barid | int | 网吧id |

| 参数名 | 参数类型 | 参数说明 | 备注 |
