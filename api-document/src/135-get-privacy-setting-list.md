# 获取隐私设置开关列表

* POST `/capi.php`
* c `135`

## v2.0 添加

## 请求参数

| 参数名 | 参数类型 | 参数说明 | 备注 |
| :---- | :----| :----| :---- |
| sid | string | sessionid | 登录成功后返回 |
| id | string | 用户id | 登录成功后返回 |

## 返回参数
| 参数名 | 参数类型 | 参数说明 | 备注 |
| :---- | :----| :----| :---- |
| data | object array | 隐私设置配置列表 |  |

## data
| 参数名 | 参数类型 | 参数说明 | 备注 |
| :---- | :----| :----| :---- |
| e_name | string | 配置选项的英文名称 |  |
| c_name | string | 配置选项的中文名称 |  |
| value | int | 配置选项对应的设置状态 | 1 开，0 关 |

## 返回实例

```JSON
{
  "status": 1,
  "info": "",
  "data": [
    {
      "e_name" : "no_wanjia",
      "c_name" : "不出现在\"玩家\"列表",
      "value" : 1
    }
  ]
}
```
