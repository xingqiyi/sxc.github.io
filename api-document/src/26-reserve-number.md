# 用户当天订座次数

* POST `/capi.php`
* c `26`

## 请求参数

| 参数名 | 参数类型 | 参数说明 | 备注 |
| :---- | :----| :----| :---- |
| sid | string | sessionid | 登录成功后返回 |
| id | string | 用户id | 登录成功后返回 |

## 返回参数

| 参数名 | 参数类型 | 参数说明 | 备注 |
| :---- | :----| :----| :---- |
| data | int | 用户当天订座次数 |

## 返回实例

```JSON
{
  "status": 1,
  "info": "",
  "data": "3"
}
```
